此方法试验成功的请情况是在使用Emberjs框架的时候试验成功的。其实node的升级和EmberJs的关系不是很大，操作很类似罢了。 下载linux二进制64位包，放在usr/local/src目录下，编辑环境，root/.bashrc。
ember升级，通过手动修改ember的版本号。修改之后的bower.json如下：其中需要修改的版本号有三个：ember、ember-data、jquery。在使用命令更新之前最好先删除原有的文件。
* 删除bower_components/ember
* 删除bower_components/ember-data
* 删除bower_components/jquery。
需要重新执行npm、bower命令下载最新的文件和依赖文件。
npm install bower install
